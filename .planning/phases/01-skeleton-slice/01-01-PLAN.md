## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → nginx (port 3000) | Public host port; untrusted user input crosses here |
| nginx → backend (compose-internal :8000) | Backend not host-exposed (INFRA-04); trust raised by topology |
| Container filesystem → host bind-mounts (`./data`, `./config`) | Container root user writes to host-owned dirs (Pitfall #14, accepted for v1) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | DoS | `EstimateRequest.pages` | mitigate | `Field(ge=0, le=10_000)` in `backend/app/schemas/estimate.py` — prevents `pages=10**18 × 4.0` Edge-Case. Per RESEARCH §Security "Phase-1-Härtungs-Empfehlung". |
| T-01-02 | Tampering | `complexity` JSON field | mitigate | `Literal["low","medium","high","very_high"]` in schema; Pydantic auto-422 on any other value. |
| T-01-03 | Information Disclosure | Backend port 8000 leaked to host | mitigate | `expose: ["8000"]` only in `docker-compose.yml` (Plan 03), NOT `ports:`. Smoke test in Plan 03 verifies `curl localhost:8000` fails. |
| T-01-04 | Information Disclosure | OpenAPI docs on `/api/docs` | accept | No auth in v1 by design; documented for review in Phase 3 (when `get_current_user_optional()` stub lands, docs_url can be gated). |
| T-01-05 | Spoofing | `proxy_set_header Host $host` forwards user-supplied Host header | accept | Backend has no host-allowlist or absolute-URL generation in Phase 1; revisit in Phase 4 when WeasyPrint may render asset URLs. |
| T-01-06 | Information Disclosure | Pydantic validation errors leak internals | accept | FastAPI default returns structured JSON without stack traces; sufficient for v1. |
| T-01-SC | Tampering | uv-installed PyPI packages (fastapi, uvicorn, pydantic, transitive deps) | accept | slopcheck audit in 01-RESEARCH.md §"Package Legitimacy Audit" verified all 10 direct deps `[OK]`. No `[ASSUMED]`/`[SUS]` packages in this plan. |
